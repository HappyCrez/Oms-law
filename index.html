Закон Ома

  • В таблице указаны значения показателей приборов: Амперметра, Вольтметра и длина проводника.
  • Для выставления длины проводника, необходимо зажать левой кнопкой мыши по черному прямоугольнику под надписью "Проводник с подвижным контактом", а затем потянуть мышку влевую (для уменьшения длины проводящей зоны) или вправую (для увеличения длины проводящей зоны) сторону. Значение длины при каждом изменении будет обновляться в таблице
  • Для управления потенциометром необходимо зажать левую кнопку мыши вокруг него (или на нём самом), а затем потянуть вращательным движением, по часовой стрелке для увеличения сопротивления, против часовой для уменьшения. При прохождении полного оборота сопротивление начинает отсчёт с нуля!
  • Для начала работы, установку необходимо включить (нажать на зеленую кнопку под схемой)
  • Первая таблица
  • Необходимо выставить длину проводника - 50см
  • 1. Выставить сопротивление с помощью магазина сопротивлений, чтобы сила тока стала равна 0.198А
  • 2. Записать значения силы тока и напряжения в таблицу (таблица дана в методических указаниях)
  • 3. Каждый раз, уменьшая длину проводника на 5см, повторять действия пунктов 1-3 (всего 10 измерений)
  • Вторая таблица
  • Необходимо выставить длину проводника в интервале 40-50см
  • 1. Изменять значение силы тока в цепи с помощью потенциометра, соблюдая равномерное увеличение
  • 2. Записывать значения силы тока и напряжения в таблицу (таблица дана в методических указаниях)
  • Данные установки:
  • Вольтметр, класс точности 1.5, предельное значение 1.5В
  • Амперметр, класс точности 1.5, предельное значение 0.25А
  • Линейка, систематическая ошибка - 0.5мм
  • Микрометр, систематическая ошибка - 0.01мм
  • Диаметр проволки (измерения микрометром) 0.31мм, 0.30мм, 0.31мм, 0.32мм, 0.31мм
